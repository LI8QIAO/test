# 🎮 反直觉反应力综合挑战 (Intuition.X)

这是一个基于认知心理学原理开发的网页测试游戏，旨在通过制造各种“认知冲突”来挑战玩家的本能直觉和肌肉记忆。

## 🚀 项目特性

- **多维度测试**：包含 5 个精心设计的反常识挑战项目。
- **动态视觉**：基于 Framer Motion 的丝滑动画与全局低沉炫彩渐变背景。
- **高精度反馈**：毫秒级反应时间记录与实时正确性反馈。
- **健壮逻辑**：采用物理锁（Ref）机制，杜绝重复触发与状态冲突。

## 📋 五大核心项目

1. **颜色 Stroop**：挑战自动阅读本能，点击文字的物理颜色。
2. **方向矛盾**：视觉朝向与文字指令的博弈，箭头往往是谎言。
3. **质合干扰**：在巨大的字体干扰下，快速判断数字的数学属性（质数/合数）。
4. **镜像映射**：打破身体同侧映射，左手图点右，右手图点左。
5. **语义翻转**：强制逆转情绪效价，正面词需做出负面反应。

## 🛠️ 技术栈

- **React 18**：前端 UI 框架。
- **Tailwind CSS**：响应式样式与现代 UI 设计。
- **Framer Motion**：高性能动画引擎。
- **Lucide React**：精美图标库。
- **Vite**：极速构建工具。

## 📦 如何运行

1. **克隆项目并安装依赖**：
   ```bash
   npm install
   ```
2. **启动开发服务器**：
   ```bash
   npm run dev
   ```
3. **访问游戏**：
   打开浏览器访问 `http://localhost:5173/`。

## 🌐 部署说明

项目已配置为支持一键部署至 **Vercel** 或 **Netlify**。同时提供单文件独立版（standalone.html），无需任何环境即可双击游玩。
